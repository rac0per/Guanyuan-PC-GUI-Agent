# 智能 Agent 创新大赛

## 赛题4：PC GUI Agent框架

### 赛题描述

GUI Agent（图形用户界面智能体）正在成为连接现实与数字世界的桥梁，一些开源项目如通义NLP实验室的MobileAgent项目，构建了视觉感知方案的自动化设备操作智能体。但在面对生产生活场景中复杂任务时，仅依靠点击、输入等键鼠操作，可能会限制Agent的任务完成效率和成功率。本赛题要求参赛者设计、实现面向PC（个人电脑）的GUI Agent框架，该框架能够结合GUI操作、代码生成或工具调用，提升PC软件使用、Web浏览等场景的任务完成效率和成功率。

### 技术要求与架构参考

参赛者应借鉴最新的GUI Agent学术研究和工程实践，设计方案可以参考以下内容（不限于）：

- Agent架构：架构设计可参考现有GUI Agent框架，通过分离规划（planner）、执行（worker）、反思（reflector）和记忆（memory）等模块，实现任务分解和多模型协作。

- 动作空间：Agent的动作空间不局限于点击、输入等键鼠操作，可以直接根据当前任务生成代码，也可以从预定义或动态加载的MCP工具库中选择工具，例如在Chrome中打开指定url、在VS Code中安装指定插件、在Excel表格指定单元格中填入内容等，以提升在PC端复杂场景的能力。

- 基础模型：限制使用开源模型。
